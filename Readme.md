
# Talent IQ

Talent IQ is a coding platform where users can create or join live coding sessions and solve problems together.

## Features
- User login and signup
- Create coding sessions
- Join live sessions
- Solve coding problems
- Code editor with output
- Video and chat during session
- Dashboard to see active and past sessions

## Tech Used
Frontend:
- React
- Vite
- Tailwind CSS
- DaisyUI

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

Other Tools:
- Clerk (authentication)
- Stream (video and chat)
- Inngest (background events)

## Run Project

Backend
```bash
cd backend
npm install
npm run dev
````

Frontend

```bash
cd frontend
npm install
npm run dev
```
